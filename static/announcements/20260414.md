# New Store Format

With the release of client v12.1.1, Star Trek: Timelines debuted a new store layout. This was a structural change in the
way the store is organized, internally, and not simply a cosmetic update.

For this reason, the "Current Offers" tab on the main site (datacore.app) no longer reflects all of the available offers
in the game.

The beta site introduces support for the new format. If you consult the Offers tab regularly, use [Betacore](https://beta.datacore.app) until the next release.
